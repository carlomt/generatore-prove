# generatore-prove
Codice per creare delle prove in pdf a partire da un elenco di domande (con le relative risposte) in formato CSV o Excel.

Si aspetta le domande sulla prima colonna e le risposte sulle colonne successive. Si può usare scaricandolo (richiede python e LaTeX [![o direttamente online:](https://img.shields.io/badge/Open-App-brightgreen)](https://generatore-prove-production.up.railway.app/)
